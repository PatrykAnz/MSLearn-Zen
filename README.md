# MSLearn-Zen - Microsoft Learn Zen Mode Extension

# Unofficial extension. Not affiliated with Microsoft.

This Chrome extension makes Microsoft Learn training pages easier to read by removing clutter that doenst need to be used when learning. It also applies a custom dark theme.

## What it does

- **Toggleable Zen Mode**: Toggle on zen mode on and off
- **Removes Distractions**:
  - Header section (navigation, search, user menu)
  - Footer section (links, theme selector, privacy information)
  - Feedback section ("Was this page helpful?" section)
- **Custom Dark Theme**

## How to install

### For Chrome/Edge

1. Go to `chrome://extensions/` 
2. Turn on **Developer mode** (there is a toggle in the top right)
3. Click **"Load unpacked"**
4. Select this folder
5. Done! You should see the extension icon now

## How to use it

1. Go to any Microsoft Learn training page (like `https://learn.microsoft.com/en-us/training/modules/...`)
2. Click the extension icon in your toolbar
3. Click **"Enable Zen Mode"** to hide all the distracting stuff
4. Click **"Disable Zen Mode"** if you want everything back
